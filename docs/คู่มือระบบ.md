## 🟢 2. ระบบเปิด-ปิดยังไง

### เปิดเอง (ปกติ)
**เปิดเครื่อง → login Windows → ทุกอย่างขึ้นเอง:**
- Backend server (`:8088`) เปิดเอง (Task `AffiliateAutopilotServer`)
- Tailscale Funnel เปิดเอง (Windows Service) → URL ถาวรใช้ได้ทันที

### ถ้าเว็บเข้าไม่ได้ / server ไม่ขึ้น
เปิด PowerShell ในโฟลเดอร์ `affiliate-autopilot` แล้วรัน:
```powershell
./scripts/start.ps1
```

### อยากให้รัน 24/7
**เปิดเครื่องค้างไว้** (อย่า shutdown) — server + tunnel เปิดเองตลอด ไม่ต้องทำอะไรอีก

## 💰 3. ขั้นตอนหาเงิน (ภาพรวม)

```
ดึงร้าน → กดรัน → AI สร้างคลิป+เสียงไทย (จากรูปอาหารจริง) → โพสต์ YouTube/FB/IG
        → วางลิงก์ affiliate ในคอมเมนต์ → คนกดซื้อ = ได้คอมมิชชั่น → ดูผลในรายงาน
```

**ใช้งานจริง 3 ปุ่มหลัก** (ดูรายละเอียดในคู่มือ Dashboard):
- **🔍 ดึงสินค้า** / **+ เพิ่มร้าน** — เอาร้านเข้าระบบ
- **▶ รันครบวง** — สร้างคลิป + โพสต์ ทุกร้านใหม่
- **🤖 Auto** — ให้ระบบทำเองตามรอบ (ดึง→สร้าง→โพสต์)

## 📦 6. ย้ายไปเครื่องใหม่

1. เครื่องใหม่ลง: **Git · Python 3.11+ · ffmpeg · Chrome**
2. `git clone -b sprint2-connect-apis <repo> affiliate-autopilot`
3. ก๊อป 2 ไฟล์จากเครื่องเก่ามาวางในโฟลเดอร์: `affiliate_migrate.zip` (จำเป็น) + `affiliate_media.zip` (คลิปเก่า)
4. รันตัวเดียวจบ:
   ```powershell
   ./scripts/migrate_finish.ps1
   ```
   (แตก zip + ติดตั้ง + เปิดระบบ ให้อัตโนมัติ)
5. login Google Flow ใน Chrome ที่เด้งมา 1 ครั้ง
6. (ถ้าอยากได้ URL ถาวรบนเครื่องใหม่) ลง Tailscale + `tailscale funnel --bg 8088`

> 🔒 ย้ายเสร็จ **ลบ 2 zip ทิ้งทั้ง 2 เครื่อง** (มี key อยู่ใน migrate.zip)

## 🏗️ 8. โครงสร้างระบบ (เผื่ออยากรู้)

- **Backend** = FastAPI (Python) รัน native บนเครื่อง Windows · พอร์ต `:8088` · ฐานข้อมูล SQLite (`data/`)
- **หน้าเว็บ** = เสิร์ฟจาก backend ตรงๆ (เปิด URL เดียวได้ทั้งเว็บ+API)
- **วีดีโอ** = Google Flow (เบราว์เซอร์อัตโนมัติ) ทำคลิปจากรูปอาหารจริง + เสียงพากย์ไทย (edge-tts ฟรี)
- **เข้าจากภายนอก** = Tailscale Funnel (URL ถาวรฟรี ไม่ต้องมีโดเมน)
- **รันตลอด** = Task Scheduler (server) + Tailscale Service (tunnel) เปิดเองตอน login

> 💡 เครื่องนี้รัน **native ไม่ใช้ Docker** (RAM 8GB + ฟีเจอร์ Flow ต้องใช้ Chrome บนเครื่องจริง) — เบา เสถียร ครบทุกฟีเจอร์
