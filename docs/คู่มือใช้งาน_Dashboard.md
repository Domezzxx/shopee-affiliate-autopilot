# 🖥️ คู่มือใช้งาน Dashboard — Affiliate Autopilot

> ระบบทำคลิปรีวิวอาหาร Shopee Food + โพสต์อัตโนมัติ + วาง affiliate link หาคอมมิชชั่น
> คู่มือนี้สอน **ตั้งแต่เปิดระบบ → ใช้ทุกปุ่ม/ทุกแท็บ → ครบวงจรหาเงิน**

---

## 🟢 ส่วนที่ 1 — เริ่มต้น: เปิดระบบ

1. เปิด PowerShell ในโฟลเดอร์ `affiliate-autopilot`
2. รันคำสั่งเดียว:
   ```powershell
   ./scripts/start.ps1
   ```
   สคริปต์จะ: เปิดเซิร์ฟเวอร์ (`:8088`) + เปิด Chrome สำหรับ Google Flow (`:9222`) + เปิดเว็บให้อัตโนมัติ
3. เว็บเปิดที่ **http://127.0.0.1:8088/**
4. **ครั้งแรกครั้งเดียว:** ใน Chrome ที่เด้งขึ้น → ล็อกอิน Google Flow (labs.google/flow) — ระบบจำไว้ให้

> 💡 ปิดระบบ: `./scripts/stop.ps1` · เปิดใหม่: `./scripts/start.ps1` (ข้อมูลไม่หาย)

## 🗺️ ส่วนที่ 2 — ภาพรวมหน้าจอ (มองรวมก่อน)

หน้าจอแบ่งเป็น 4 ชั้นจากบนลงล่าง:

| ชั้น | คืออะไร |
|------|---------|
| **แถบบนสุด** (Header) | ปุ่มควบคุมหลัก: เปิด/ปิดระบบ, Auto-Pilot, ดึงสินค้า, เพิ่มร้าน, รันครบวง |
| **แถบสถานะ** (Banner) | บอกว่าอยู่โหมดจริง (🟢 REAL) หรือโหมดทดสอบ (🟡 MOCK) |
| **การ์ดตัวเลข** (KPIs) | สรุปภาพรวม: ร้าน, โพสต์, รายได้, CTR ฯลฯ |
| **แท็บ** (Tabs) | 6 แท็บ: ร้านค้า · คอนเทนต์ A/B · โพสต์ · ช่องทาง · รายงาน · Flow ระบบ |

## 🎛️ ส่วนที่ 3 — ปุ่มบนแถบบนสุด (Header)

| ปุ่ม | ทำอะไร |
|------|--------|
| **● ระบบ** | สวิตช์ใหญ่ เปิด/ปิดทั้งระบบ · 🟢 เปิด = บอททำงานได้ / 🔴 ปิด = หยุดทุกอย่าง · **ชี้เมาส์ค้าง** เห็นสถานะว่าอะไรพร้อม/ไม่พร้อม |
| **🤖 Auto** | เปิด/ปิด **Auto-Pilot** — ให้ระบบดึงร้าน+สร้างคลิป+โพสต์ **เองตามรอบ** โดยไม่ต้องกดเอง |
| **🔗 เชื่อมต่อ API** | (สำหรับใช้ผ่านมือถือ/รีโมท) ใส่ URL ของเครื่องที่รันบอท · ใช้ในเครื่องเดียวกัน **ไม่ต้องตั้ง** |
| **🌐 เปิด Google Flow** | เปิด Chrome ของบอทสำหรับสร้างวีดีโอ (ถ้าปิดไป) |
| **🔍 ดึงสินค้า** | ดึงร้าน/สินค้าจาก Shopee ตามคำค้นใน `.env` เข้าระบบ |
| **+ เพิ่มร้าน** | เปิดฟอร์มเพิ่มร้านเอง (หรืออัปโหลด CSV) |
| **▶ รันครบวง** | สั่งระบบ **สร้างคลิป + โพสต์** ให้ทุกร้านที่ยังใหม่ในคำสั่งเดียว |
| **เครื่องมือ ▾** | 🚦 เช็คพร้อมโพสต์ (เช็ค FB/IG/YouTube พร้อมไหม) · ⚙ Auto-optimize (หยุดร้าน CTR ต่ำ) · 🎲 จำลองผล (ใส่ตัวเลขเดโม) |

## 📊 ส่วนที่ 4 — การ์ดตัวเลข (KPIs) อ่านอย่างไร

| การ์ด | ความหมาย |
|-------|----------|
| **ร้านทั้งหมด** | จำนวนร้านในระบบ · เขียว = active (ทำงาน) / แดง = paused (หยุด) |
| **โพสต์** | จำนวนโพสต์ที่ยิงออกไป · แดง = ล้มเหลวกี่อัน |
| **รายได้ประเมิน** | ประเมินจากคลิก × ค่าคอมเฉลี่ย · ด้านล่าง = **กำไรสุทธิ** (รายได้ − ต้นทุน AI) |
| **Impressions** | ยอดคนเห็นรวม |
| **CTR เฉลี่ย** | อัตราคนกดลิงก์ (ยิ่งสูงยิ่งดี — คลิปปัง) |
| **ค่า AI เขียน (สะสม)** | ต้นทุนค่า AI เขียนคอนเทนต์รวม |
| **สถานะระบบ** | ● = เชื่อมแล้ว / ○ = ยังไม่เชื่อม (Claude/Gemini/Meta) + โหมดโพสต์ + วีดีโอเปิด/ปิด |

## 📑 ส่วนที่ 5 — แท็บทั้ง 6 (หัวใจการใช้งาน)

### 1️⃣ แท็บ "ร้านค้า"
หน้าหลักจัดการร้าน — แต่ละร้านเป็นการ์ด แสดง: ชื่อ · ย่าน · ⭐เรตติ้ง · เมนู · มีลิงก์ affiliate ไหม · ทุน/กำไร
- ☑️ **ต้องการอนุมัติก่อนโพสต์** — ติ๊ก = สร้างคลิปเสร็จต้องกดอนุมัติก่อนถึงจะโพสต์ (กันโพสต์พลาด)
- ▶ **รันร้านนี้** — สร้างคลิป + โพสต์ ร้านนี้ทันที
- **+ เพิ่มร้าน** (ปุ่มบน) → ฟอร์มกรอกเอง หรือ **📄 อัปโหลด CSV** ทีละหลายร้าน

### 2️⃣ แท็บ "คอนเทนต์ A/B"
ดูคลิป/คอนเทนต์ที่ AI สร้าง (ต่อร้าน) — มี **2 แบบ A/B** เทียบกันต่อแพลตฟอร์ม
- พรีวิววีดีโอ/ภาพแต่ละ variant + hook + 💬 คอมเมนต์แรก (มี affiliate link)
- ปุ่ม: 👩/👨 เลือกเสียง · 🎬 **รวมคลิป A/B** (ทำคลิปรวม) · 🍜 **รีวิวในร้าน** (พ่อครัวพูด)
- ถ้าร้านตั้ง "ต้องอนุมัติ" → มีปุ่ม ✅ **อนุมัติ & โพสต์**
- 🔎 ตัวกรอง: เดือน/ปี + ช่องทาง
- 🏆 ผล A/B: โชว์ CTR ของ A vs B ว่าตัวไหนชนะ

### 3️⃣ แท็บ "โพสต์"
ตารางประวัติการโพสต์ทั้งหมด: **เวลา · Platform · วิธี (api/phone) · บัญชี · สถานะ · คอมเมนต์ลิงก์ · ID**
- สถานะ: `posted` (สำเร็จ) / `queued` (รอ) / `failed` (พลาด)
- 🔎 ตัวกรอง: เดือน · วันที่ · ช่องทาง · สถานะ
> 👉 อยากรู้ว่า "โพสต์ขึ้น YouTube จริงไหม" ดูที่แท็บนี้

### 4️⃣ แท็บ "ช่องทาง"
สรุปผลแยกตามแพลตฟอร์ม (Facebook / Instagram / YouTube): **CTR · Impressions · Clicks** — ดูว่าช่องไหนเวิร์กสุด

### 5️⃣ แท็บ "📊 รายงาน"
รายงานสรุปรายวัน: ร้าน active/หยุด · โพสต์สำเร็จ/พลาด · Impressions · CTR · รายได้ · ต้นทุน
+ ตาราง 🏆 **ผู้ชนะ A/B ต่อร้าน** (ตัวไหนดีกว่ากี่ %)

### 6️⃣ แท็บ "Flow ระบบ"
แผนผังอธิบายระบบทำงาน 5 ขั้น: ดึงร้าน → AI เขียน → AI ทำสื่อ → โพสต์อัตโนมัติ → เก็บผล+optimize

---

## 🚀 ส่วนที่ 6 — ขั้นตอนใช้งานครั้งแรก (ทำตามนี้)

1. **เปิดระบบ** → `./scripts/start.ps1` → เว็บเด้งขึ้น
2. เช็ค **แถบ Banner** ต้องเป็น 🟢 REAL MODE (ถ้า 🟡 MOCK = ยังไม่ใส่ Gemini key ดูคู่มือ API)
3. กด **● ระบบ** ให้เป็น 🟢 ระบบเปิด
4. **ได้ร้านเข้าระบบ** — เลือกทางใดทางหนึ่ง:
   - กด **🔍 ดึงสินค้า** (ดึงจาก Shopee อัตโนมัติ) หรือ
   - กด **+ เพิ่มร้าน** กรอกเอง / อัปโหลด CSV
5. ไปแท็บ **ร้านค้า** → กด **▶ รันร้านนี้** (หรือ **▶ รันครบวง** ทำทุกร้าน)
6. ดู **แถบ Progress** ด้านบนวิ่ง (AI กำลังสร้างคลิป)
7. เสร็จแล้วดูคลิปที่แท็บ **คอนเทนต์ A/B** → ตรวจคลิป + เสียงไทย
8. เช็คการโพสต์ที่แท็บ **โพสต์** (ขึ้น YouTube/FB/IG หรือยัง)
9. ผ่านไปวันสองวัน ดูผลที่แท็บ **รายงาน** + **ช่องทาง** ว่าคลิปไหนปัง

> ⚡ **อยากให้ทำเองทั้งหมด:** กด **🤖 Auto** เปิด Auto-Pilot — ระบบจะดึงร้าน+สร้าง+โพสต์เองตามรอบ ไม่ต้องนั่งกด

## 🧰 ส่วนที่ 7 — งานที่ทำบ่อย

| อยากทำ | ทำยังไง |
|--------|---------|
| เพิ่มร้านใหม่ | **+ เพิ่มร้าน** → กรอก → "บันทึก + รันร้านนี้" |
| เพิ่มทีหลายร้าน | **+ เพิ่มร้าน** → **📄 อัปโหลด CSV** |
| สร้างคลิป+โพสต์ร้านเดียว | แท็บ ร้านค้า → **▶ รันร้านนี้** |
| สร้าง+โพสต์ทุกร้านใหม่ | **▶ รันครบวง** |
| เช็คก่อนโพสต์ว่าพร้อมไหม | เครื่องมือ ▾ → **🚦 เช็คพร้อมโพสต์** |
| โพสต์ต้องอนุมัติก่อน | ติ๊ก "ต้องการอนุมัติ" ที่ร้าน → คอนเทนต์ A/B → **✅ อนุมัติ & โพสต์** |
| ดูว่าโพสต์ขึ้นจริงไหม | แท็บ **โพสต์** (สถานะ posted) |
| ให้ระบบทำเองตลอด | **🤖 Auto** เปิด Auto-Pilot |

## 🎨 ส่วนที่ 8 — สี/สถานะ ที่ต้องรู้

- 🟢 เขียว = ดี/เปิด/สำเร็จ (active, posted, REAL mode)
- 🔴 แดง = หยุด/ปิด/ล้มเหลว (paused, failed)
- 🟡 เหลือง = โหมดทดสอบ / รออนุมัติ / กำลังเก็บข้อมูล
- ● ทึบ = เชื่อมแล้ว / ○ กลวง = ยังไม่เชื่อม

## 🆘 ส่วนที่ 9 — แก้ปัญหาเบื้องต้น

| อาการ | สาเหตุ/วิธีแก้ |
|-------|---------------|
| Banner เป็น 🟡 MOCK | ยังไม่ใส่ `GEMINI_API_KEY` ใน `.env` (ดูคู่มือ API) |
| สร้างคลิปแล้วไม่โพสต์ YouTube | "สร้าง" กับ "โพสต์" คนละขั้น — ใช้ **▶ รันร้านนี้/รันครบวง** (สร้าง+โพสต์) หรือเปิด Auto · เช็คว่าร้านไม่ติด "ต้องอนุมัติ" |
| วีดีโอไม่สร้าง | เช็ค **🌐 เปิด Google Flow** (Chrome bot) + ล็อกอิน Flow + `ENABLE_VIDEO=true` |
| เขียนคอนเทนต์ error 429 | Gemini เครดิตหมด → เติมที่ ai.studio |
| กดปุ่มแล้วเด้ง error แดง | ระบบปิดอยู่ → กด **● ระบบ** ให้เป็น 🟢 ก่อน |
